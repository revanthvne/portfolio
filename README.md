# Portfolio Website

**Personal Portfolio — Revanth Kondaveti**

A single-file, zero-dependency portfolio website with dark theme, particle backgrounds, SVG architecture diagrams, scroll-reveal animations, and animated gradient borders.

## Features

- **Particle Background** — HTML5 Canvas with connected particle network
- **SVG Architecture Diagrams** — Animated diagrams for each project
- **Animated Gradient Borders** — CSS @property rotating conic-gradient borders
- **Scroll Reveal** — IntersectionObserver-powered animations
- **Animated Counters** — Stats count up on scroll
- **Timeline Layout** — Interactive work experience timeline
- **Responsive Design** — Mobile-first responsive layout
- **Zero Dependencies** — Single HTML file

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- HTML5 Canvas, SVG, CSS Animations
